RALPH ANIME THE EXPLORER

Credits: Data from Jikan API (MyAnimeList unofficial API).

API Info (Project Requirements)

Base URL: https://api.jikan.moe/v4/

Endpoints used:

  • GET /top/anime?filter=bypopularity&limit=12
  • GET /anime/{id}
  • GET /anime?q={query}&limit=12
  • GET /anime/{id}/characters
  • GET /anime/{id}/episodes

Required parameters: {id} (path), q, limit, filter (query).

Authentication: None (no API key required).
